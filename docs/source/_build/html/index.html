DoiWorks documentation

DoiWorks is a protocol and a set of tools for fraud-resistant verification and documentation of Double Opt-In´s in email marketing environments on the Doichain blockchain, http://www.doichain.org, an open, decentralized and permissionless merged-mining PoW-blockchain.

Note

This project is under active development.
